# TaskPlayon Invoice Management

A simple **Invoice Management System** built with **ASP.NET Core MVC** for managing customers, products, and invoices with PDF generation.

---

## Features

- Add and manage **Customers** and **Products**
- Create and **edit invoices** with multiple products
- Auto-calculate **line totals** and **grand total**
- Generate **PDF invoices** for download
- **Data seeding** for initial products and customers
- **Audit logs** for create, update, delete operations

---

## Technologies

- ASP.NET Core MVC (.NET 9)
- Entity Framework Core
- Bootstrap 5, jQuery
- SweetAlert2 for notifications

---

## Setup

1. Clone repo:
```bash
git clone https://github.com/rajdip27/TaskPlayon.git
cd TaskPlayon
````

2. Restore packages:

```bash
dotnet restore
```

3. Update database:

```bash
dotnet ef database update
```

4. Run app:

```bash
dotnet run
```

---

## Usage

* Navigate to **Product** or **Invoice**
* Add customers/products to invoice
* Click **Save Invoice**
* Generate **PDF invoice**
